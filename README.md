#### Minecraft Clicker
Minecraft Clicker is an unofficial and non-profit Minecraft themed clicker game I made as a project for fun. You click the grass block to gain grass. Spend this grass on upgrades to unlock boosts such as increased grass per click, and an autoclicker. Use these boosts to get more grass. Once you have enough grass, you can prestige, and get a boost on your clicks and autoclicker (this wont show in stats). Have fun!
